# WP Blueprints

A collection of WordPress Playground Blueprints for repeatable site, plugin, theme, and training environments.

## Samples

* [Plugin Developer Lab](blueprints/plugin-developer-lab/README.md) - Query Monitor, Debug Bar, log viewing, debug constants, and a sample plugin scaffold.

## Tools

* [WP Blueprint Exporter](plugins/wp-blueprint-exporter/README.md) - WordPress admin plugin for exporting active plugins, active theme, site options, and login setup as a Blueprint.
